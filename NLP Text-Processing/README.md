# NLP Text-Processing

This folder contains a collection of beginner-friendly Natural Language Processing (NLP) text preprocessing notebooks. Each notebook focuses on a core preprocessing technique commonly used before applying machine learning or deep learning models on text data.

These notebooks are designed for learning, practice, and quick reference, and are implemented using popular Python NLP libraries such as **NLTK**.

---

## üìå Topics Covered

- **Tokenization**
  - Sentence tokenization
  - Word tokenization

- **Stopword Removal**
  - Removing common words that do not contribute much semantic meaning

- **Stemming**
  - Rule-based word reduction techniques
  - Different types of stemming algorithms

- **Lemmatization**
  - Converting words into their base or dictionary form

- **Parts of Speech (POS) Tagging**
  - Assigning grammatical tags to words (noun, verb, adjective, etc.)

- **Named Entity Recognition (NER)**
  - Identifying entities such as names, locations, organizations, and dates
